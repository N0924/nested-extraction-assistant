# 嵌套解压助手 v0.2.1

这是 Windows 64 位便携版的启动兼容性修复版本。

## 本次修复

- 修复程序位于多层中文目录时，Tkinter 找不到 Tcl/Tk 运行库并导致启动失败的问题；
- 恢复 PyInstaller 标准的 `_internal\_tcl_data` 和 `_internal\_tk_data` 路径；
- 删除便携包中重复的顶层 Tcl/Tk 运行库副本。

## 使用前准备

1. 下载 `嵌套解压助手-v0.2.1-Windows-x64.zip`；
2. 完整解压 ZIP，并保留 EXE 旁边的 `_internal` 目录；
3. 确认 Windows 已安装 WinRAR；
4. 双击 `嵌套解压助手.exe`，在设置中确认 WinRAR、工作目录和输出目录。
