# 嵌套解压助手 v0.2.0

这是首次公开发布的 Windows 64 位便携版。

## 主要功能

- 递归处理普通压缩包、错误扩展名文件、嵌入式 ZIP 和 Apate 兼容伪装；
- 识别常见 RAR、7Z、ZIP 分卷和连续编号分卷；
- 每个任务可设置独立密码，留空时按密码池顺序尝试；
- 多任务队列、拖拽导入、任务历史和全局/单任务日志；
- 源文件只读，工作副本和最终输出彼此隔离。

## 使用前准备

1. 下载 `NestedExtractionAssistant-v0.2.0-win64.zip`；
2. 完整解压 ZIP，不能只取出 EXE；
3. 确认 Windows 已安装 WinRAR；
4. 双击 `嵌套解压助手.exe`，在设置中确认 WinRAR、工作目录和输出目录。

建议先用容量较小、能够重新取得的文件验证。程序不会打包或调用 `apate.exe`，也不包含 WinRAR。
