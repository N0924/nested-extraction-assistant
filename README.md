# 嵌套解压助手

[简体中文](README.md) | [English](README-英文.md)

一款面向 Windows 的本地解压工具，用于处理多层压缩包、错误后缀、伪装文件和常见分卷。

![嵌套解压助手主界面](assets/主界面.png)

## 主要功能

- 自动识别并解压多种伪装压缩包格式；
- 支持压缩包嵌套，自动处理解压后的内层压缩包；
- 支持错误后缀、嵌入式 ZIP 和 Apate 兼容伪装；
- 支持常见 RAR、ZIP、7Z 分卷及连续数字分卷；
- 每个任务可以单独填写密码，也可以按顺序尝试密码池；
- 支持拖入文件或文件夹，并通过任务列表查看处理进度和历史；
- 源文件保持不变，工作文件和最终结果保存到独立目录。

## 使用条件

- Windows 10 或 Windows 11，64 位系统；
- 电脑已安装 WinRAR；
- 使用便携版不需要安装 Python。

## 快速开始

1. 完整解压 ZIP，不要只取出其中的 EXE；
2. 双击 `嵌套解压助手.exe`；
3. 在设置中确认 WinRAR、工作目录和输出目录；
4. 添加或拖入文件，根据需要填写任务密码或密码池，然后开始解压。

工作目录和输出目录应彼此独立，也不要放在源文件目录中。
